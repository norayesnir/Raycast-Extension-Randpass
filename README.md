# Random Password Generator

Generate secure passphrases using the EFF Large Wordlist.

## Features

- Generates passwords in format: `[special+number]Word-Word-Word-Word[special+number]`
- Uses EFF's large wordlist (7,776 words)
- Automatically copies to clipboard
- No dependencies on external binaries (Go, etc.)
- Works cross-platform (macOS, Windows, Linux)

## Usage

Run the "Generate Password" command from Raycast.

## Example Output

`@4Abolish-Jelly-Snowman-Thermal7%`
